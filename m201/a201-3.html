1. Los elementos web tienen un diseño por defecto

2. Podemos modificar el diseño de los elementos web a través de reglas de diseño

El lenguaje CSS (Cascade-Style-Sheet) define reglas en cascada (que se heredan), para determinar cuál será el diseño de un elemento específico.

3. Podemos Especificar las reglas de diseño desde otro archivo mediante la etiqueta <link> en las cabeceras

            
                <head>
                    <link rel="stylesheet" href="style.css">
                </head>

4. Podemos Especificar las reglas de diseño desde otro archivo mediante la etiqueta <style> en las cabeceras

            
                <head>
                    <style>
                        #my-input-2 {
                            border: 4px dotted red;
                        }
                        
                        #my-image-2 {
                            border-radius: 10px;
                            border: 2px solid blue;
                        }
                    </style>
                </head>

5. Podemos Especificar las reglas de diseño en la etiqueta usando el atributo style

            
                <input style="border: 4px solid gray; color: red;" type="text" placeholder="Escribe algo...">
                <img style="border-radius: 50%" src="https://placekitten.com/250">
